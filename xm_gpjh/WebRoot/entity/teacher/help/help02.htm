首先确定一下你所用的系统是 windows2000/ XP 。因为我们这个上传文件的 ftp 工具需要 java 虚拟机的支持，所以在使用之前必须安装 java 虚拟机。 windows 2000 在安装时会自动提示安装 java 虚拟机，而 windows XP 不会，这样就需要你手动安装了。

For XP 的 java 虚拟机下载

点击下载安装。然后请打开 IE 工具栏上的“工具”，选择“ internet ”选项，点击“高级”选项卡，然后选择 microsoft VM 中的选项：“启用 Java JIT 编译器”“启用 Java 记录”“启用 Java 控制台”。操作如下图所示：

红框框出来的为要勾选的选项。
